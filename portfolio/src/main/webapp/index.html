My Projects

  • Socket Programming: Base-64 Encoding System

    Developed a Base64 Encoding Communication Protocol between server and client to communicate with each other based on TCP Sockets. Implemented a concurrent server to accept connections from multiple clients and serve all of them concurrently. The client and server interact using three types of messages: Type 1(Encoded text message), Type 2(ACK message), Type 3(Connection Termination message)

  • Accelerated Residual Descent Method

    Built a neural network optimization algorithm by modifying Nesterov's library. Implemented Accelerated Residual Method which improves upon time complexity and convergence rate of Conjugate Gradient Method and Nesterov's algorithm for dense matrices. Introduction of residual descent restarting technique and incorporating preconditioning techniques to accelerate its convergence.
